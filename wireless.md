### Fundamentos de Rede Wireless-Fidelity (Wi-Fi)
    Estudo básicos para entender como funcionam as redes.

### O que é SSID
    SSID (Service Set Identifier) nada mais é que o "nome técnico" da rede Wi-Fi, hosteada por um ponto de acesso. É um identificador único para diferenciá-la das Redes próximas, possuem limite de 32 caracteres e facilita a conexão dos dispositivos.

### O que é Access Point
    Acess Point (AP), ponto de acesso, é um dispositivo que amplia a área de conexão de uma rede Wi-fi, fazendo uma extensão do sinal do roteador. Ele em si, não gerencia a conexão, mas aumenta o raio (amplitude) do sinal do roteador. Seu uso é ideal em empresas e locais com alto tráfego de internet.

### Como funciona autenticação
    Verificação de credenciais (senha, certificado) para permitir o acesso à rede. Sendo o mais comum WPA2-PSK (senha compartilhada), normalmente usado residencialmente. E em empresas é usada a autenticação WPA2/WPA3-Enterprise (credenciais individuais) para cada colaborador que utiliza a rede empresarial.

### Diferença entre 2.4GHz e 5GHz
    A diferença entre as frequências da rede é sumariamente ligada à velocidade de trans ferência de arquivos, enquanto a rede 5GHz transfere arquivos mais rápido, a de 2.4GHz transfere mais lentamente. Em contrapartida, as ondas da 2.4GHz tem maior alcance que a de 5GHz.

### Principais protocolos de segurança de rede
    TLS/SSL (Transport Layer Security/Secure Sockets Layer)
        Protocolos essenciais para navegação segura na web e comunicação online. Criptografa dados entre cliente e servidor.

    IPsec (Internet Protocol Security)
        Utilizado para proteger a conexão a nível de IP, normalmente utilizado em VPNs (Virtual Private Networks).

    SSH (Secure Shell)
        Garante acesso e gerenciamento remoto de dispositivos.

    SFTP (SSH File Transfer Protocol)
        Transferência segura de arquivos, utilizando Secure Shell (SSH) para criptografia.

    DNSSEC (Domain Name System Security Extensions)
        Autentica as respostas do DNS, prevenindo manipulações externas.

    HTTPS (Hypertext Transfer Protocol Secure)
        A versão segura do HTTP (protocolo de navegação, Tim Berners-Lee), usando TLS (Transport Layer Security). É um protocolo criptográfico que garante segurança, privacidade e integridade dos dados.) para garantir que os dados não sejam interceptados.

    SNMPv3 (Simple Network Management Protocol)
        Baseado em níveis de segurança.
        É a versão mais seura do protocolo de gerenciamento de rede, oferece autenticação de usuário, criptografia de dados e integridade de mensagens (SHA/MD5 e DES/AES)
            Autenticação por meio de SHA e MD5
            Criptografia de dados com DES e AES
